Sobre Nós

"Este site é uma página fictícia, desenvolvida pelo aluno Wanderson Brito como parte do projeto de estudo nº 3 sobre Bootstrap, realizado na instituição EBAC. O objetivo deste trabalho é praticar e demonstrar os conhecimentos adquiridos no uso do framework Bootstrap para a criação de interfaces responsivas e modernas. Todas as imagens utilizadas no site foram extraídas aleatoriamente do Google Imagens, estritamente para fins educativos."
